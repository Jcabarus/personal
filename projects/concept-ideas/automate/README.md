# Automate

## Installation
Note: <br>
- If you need redownloading libraries, you will need to keep the src/images.
- Run this to install:
    - Windows: 
        - `py -m venv src/`     
        - `pip3 install pyautogui`
    - Linux/MacOS: 
        - `python3 -v venv src/` 
        - `pip3 install pyautogui`

Windows: <br>
 - Download Python on official website
 - `git clone <repository>`
 - `<activate venv>`

Linux/MacOS: <br>
 - `brew install python3`
 - `git clone <repository>`
 - `source src/bin/activate`
